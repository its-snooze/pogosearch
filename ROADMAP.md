## ğŸ“… Short Term (Next 1-2 Months)

### Premade Search Expansion
- **Trade Fodder Categories**
  - Low IV non-favorites for general trading
  - Regional exclusives for distance trades
  - Old Pokemon (2016-2020) for lucky trade candidates
  - Distance trade targets (100km+, 1000km+)

- **Storage Management Searches**
  - Low IV legendaries (0-2 star) safe to trade/transfer
  - Duplicate species checker (count2-)
  - Pokemon with no PvP/PvE value (community-defined trash list)

- **Team Rocket Counter Lists**
  - Best counters for common Rocket grunt lineups
  - Leader counter teams (Cliff, Sierra, Arlo)
  - Giovanni counter teams

- **Raid Counter Templates** 
  - Best counters for current T5 raid bosses
  - Update monthly with new raid rotations
  - Sources: LeekDuck, GamePress, PoGo Hub
  - Mega raid counters

- **PvP Quick Searches**
  - Complete PvP IV spread presets (0/15/15, 1/15/14, etc.)
  - Great League meta (top 25-50 Pokemon)
  - Ultra League meta (top 25-50 Pokemon)
  - Common spice picks for each league
  - Budget-friendly PvP options (low investment)

- **Collection Helpers**
  - Shiny checkers by generation
  - Lucky dex completion helpers
  - Shadow dex completion helpers
  - Gym defender recommendations (high CP bulky Pokemon)

### Quality of Life Improvements
- **Search History**
  - Last 10-20 generated searches
  - Quick re-apply previous searches
  - Clear history option

- **Search Sharing**
  - Generate shareable URLs with filter combinations
  - Copy link to specific search setup
  - Import search from URL parameters

- **Enhanced Import/Export**
  - Export all saved searches as JSON
  - Import search collections from file
  - Backup/restore functionality

- **Search String Validator Enhancement**
  - More detailed validation feedback
  - Suggestions for fixing common errors
  - Warning for potentially conflicting filters

### Community Features
- **Community Search Presets**
  - User-submitted search combinations (via Discord)
  - Voting/rating system for useful searches
  - Curated collection of top community searches
  - Monthly featured searches
